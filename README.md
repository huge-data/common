
# Confluent Commons

> Confluent公共工具包，包含3个库: [metrics](metrics), [config](config), [utils](utils)。


# 项目构建

## 概括

> 在运行下面命令之前，需要在主目录下运行`mvn install`，下面是分别运行每个子模块。

## Metrics模块

> 运行下面命令来构建Metrics子模块:

    $ mvn -pl :common-metrics package


## Config模块

> 运行下面命令来构建Config子模块:

    $ mvn -pl :common-config package


## Utils模块

> 运行下面命令来构建Utils子模块:

    $ mvn -pl :common-utils package
